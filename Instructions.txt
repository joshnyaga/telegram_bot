
INSTALLING PYTHON TO YOUR COMPUTER
============================================
Follow this link https://www.python.org/ftp/python/3.12.1/python-3.12.1-amd64.exe
This will download a python installer file. Install python to your computer. Follow through
the installation accepting the defaults.

Start a command prompt window as an admin
Type python --version then hit enter

incase the command does not give you  something like Python 3.11.0 which is the python version installed
know that python is not yet installed in your computer. Do not proceed with the other instructions coz they wont work

DOWNLOADING THE PROJECT
===================================================
Download the project and unzip it.
go inside the folder named telegram_bot
on the file explorer where there is the path name, delete the path name and write cmd and click enter
A command prompt window will be displayed


INSTALLING DEPENDENCIES
==============================================
on the command prompt window write pip install telethon and click enter
The process for installing the module will begin


STARTING THE APPLICATION
=====================================================
After the previous step is done, write python main.py and click enter
The program will start, prompting for API ID

OBTAINING API HASH AND API ID
==============================================
Log in to your Telegram core: https://my.telegram.org.
Go to "API development tools" and fill out the form.
You will get basic addresses as well as the api_id and api_hash parameters required for user authorization.
For the moment each number can only have one api_id connected to it.

NB
If you use the Telegram API for flooding, spamming, 
faking subscriber and view counters of channels, you will be banned forever. 


USING THE APPLICATION
===================================================
The application will prompt you to enter your API Id first, then the APi Hash and the phone number.
The application will check to see if there is any existing session and if not a code will be sent to your telegram number.
Enter the code and press enter

A menu will show. The first option 1 allows you to save all the group members of your telegram groups in a file.
The second option will allow you to send a message to all the saved group members
